# MgenixMember API (Clean Architecture + Dapper)

## Tech Stack
- .NET 8 Web API
- Clean Architecture
- Dapper
- SQL Server
- AutoMapper

## Project Structure
- API → Controllers
- Application → Services, DTOs
- Domain → Entities, Interfaces
- Infrastructure → Dapper, Repositories

## Features
- User Registration API
- Clean Architecture Implementation
- Stored Procedure with Dapper

## How to Run
1. Clone repo
2. Update connection string in appsettings.json
3. Run project
4. Open Swagger

## API Endpoint
POST /api/user/register
